Shell介绍

Shell主要分为两种，CLI（Conmand Line Inerface，命令行接口e）和GUI（Graphical User Interface，图形用户接口），操作系统的工作原理：用户 -> Shell -> Kernal（包含Drive可驱动硬件） -> Hardware -> 执行结果反馈给用户。

Linux各个目录的作用

  • root：root的家目录；
  • /home：用户的家目录（useradd user +passwd user，增加user用户并且修改user用户的密码）；
  • /boot：存放系统的启动配置文件和内核文件；
  • /usr：软件默认安装的地方；
  • /lib：存放的是32位的函数文件；
  • /lib64：存放的是64位的执行文件；
  • /bin：存放的是普通用户能执行的命令；
  • /sbin：存放的是超级用户能执行的命令；
  • /opt：存放软件的目录；
  • /dev：存放设备文件；
  • /etc：存放配置文件；
  • /media，/mnt，/run：挂载目录；
  • /proc：存放进程文件；
  • /srv：存放资源文件；
  • /sys：存放系统资源文件；
  • /tmp：存放临时文件（重要）；
  • /var：存放日志的文件（重要）；
    Linux中的带箭头的图标同Windows一样都是快捷方式（例如：/bin，/sbin，/lib，/lib64，指向/usr中的对应的文件）。

切换用户带不带"-"

su全称叫做"Switch user"，切换用户带上"-"，表示连同用户的环境变量一起切换掉，因此，切换用户时记得带"-"。
